## 🎯 Быстрый доступ

### 📱 Для пользователей
1. Откройте [Инструкция по торговле](инструкция_торговли.html)
2. Установите удобный интервал обновления (рекомендуется 5-10 сек)
3. Выбирайте размер страницы в зависимости от интернета (20-50 записей)
4. Экспортируйте сделки в конце дня

### ⚙️ Для администраторов
1. Откройте [Руководство админ-панели](руководство_админ.html)
2. Проверяйте статус воркера в админ-панели
3. Создавайте и управляйте стратегиями
4. Начинайте с Demo режима для тестирования

### 🚀 Для обновления приложения
1. Откройте [Руководство по обновлению](руководство_обновление.html)
2. Выберите один из двух способов:
   - **Ручное:** `cd /app && ./update.sh` (рекомендуется)
   - **Автоматическое:** GitHub Actions (git push = auto update)

## 💾 Сохранение в PDF

Все HTML инструкции легко сохраняются в PDF:

### На компьютере (macOS/Windows)
1. Откройте HTML файл в браузере
2. Нажмите **Ctrl+P** (Windows) или **Cmd+P** (macOS)
3. Выберите "Сохранить как PDF"
4. Укажите папку для сохранения

### На сервере
1. Используйте `wkhtmltopdf` или аналогичный инструмент
2. Или откройте в браузере на сервере и сохраните через Print Dialog
