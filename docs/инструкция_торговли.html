1️⃣ Обзор таблицы торгов

Таблица торгов отображает все закрытые сделки по всем активным стратегиям. Это ваше главное окно для мониторинга производительности торгового бота.

Ключевые особенности:

  • Данные обновляются автоматически каждые 5 секунд (по умолчанию)
  • Отображение до 1000+ закрытых сделок
  • Гибкие фильтры по стратегии, типу, дате и режиму
  • Экспорт в Excel для анализа и бухгалтерии
  • Поддержка мобильных устройств

💡 Обратите внимание

В таблице отображаются только закрытые сделки. Открытые позиции можно увидеть в админ-панели под каждой стратегией.

5️⃣ Фильтры и экспорт

📋 Фильтры

Используйте следующие фильтры для быстрого поиска нужных сделок:

  • По стратегии: Выберите конкретную стратегию (например, "Grid BTC/USDT #1") для просмотра только её сделок
  • По типу: Фильтруйте по типу торговой стратегии (Grid, Scalping, DCA, Trend, Arbitrage)
  • По дате: Выберите конкретную дату или диапазон дат для анализа результатов за период
  • По режиму: Выбирайте между Demo (демо-счёт) и Live (реальный счёт)

🎯 Совет по фильтрам

Комбинируйте фильтры! Например, выберите "Grid" + "Demo" + "Сегодня" для просмотра сегодняшних демо сделок по Grid стратегии.

📊 Экспорт в Excel

Нажмите кнопку "Экспорт в Excel" для скачивания выбранных сделок в формате .xlsx файла.

  • Файл содержит все видимые колонки таблицы
  • Применяются все активные фильтры
  • Готов для анализа в Microsoft Excel или Google Sheets
  • Содержит итоговый профит и статистику

6️⃣ Полезные советы

💡 Рекомендации для оптимального использования

  • Интервал обновления: Установите 5-10 сек для оптимального баланса между свежестью данных и нагрузкой на сеть
  • Размер страницы: Используйте 20-50 записей для быстрой загрузки на мобильных устройствах
  • Ежедневный экспорт: Экспортируйте сделки в конце дня для анализа и архивирования для целей бухгалтерии
  • Сравнение стратегий: Используйте фильтр по стратегии для сравнения производительности разных подходов
  • Анализ периодов: Смотрите результаты за конкретные дни/недели/месяцы для оценки трендов
