1️⃣ Мониторинг статуса воркера

В админ-панели добавлен новый раздел "Статус воркера" для отслеживания работы торгового бота в реальном времени.

Показатель Описание Что означает
✓ Статус Индикатор работы воркера 🟢 Зелёная = работает, 🔴 Красная = не запущен
⏱️ Последнее обновление Времени прошло с последнего сигнала Если > 60 сек — воркер зависает
🔄 Циклов Количество завершённых торговых циклов Растёт → воркер активно работает
📦 Ордеров Всего обработано ордеров Показывает общее количество сделок
⚠️ Ошибки Последняя ошибка (если была) Помогает диагностировать проблемы

🚨 Важно

Если статус показывает красный индикатор или последнее обновление > 60 секунд, воркер требует перезагрузки. Обратитесь к разделу "Устранение проблем" в руководстве.

2️⃣ Управление стратегиями

В админ-панели вы можете полностью управлять торговыми стратегиями:

  • Создавать новые стратегии (Grid, Scalping, DCA, Trend, Arbitrage)
  • Редактировать параметры существующих стратегий без остановки
  • Запускать/останавливать торговлю по каждой отдельной стратегии
  • Переключаться между Demo (тестовая) и Live (реальная) режимом
  • Устанавливать Stop-loss и risk management параметры
  • Просматривать все открытые ордера и позиции

💡 Совет

Начните с Demo режима для тестирования новых стратегий. Переходите на Live только когда убедитесь в корректности параметров.

3️⃣ Ключевые параметры стратегии

При создании или редактировании стратегии заполняйте следующие параметры:

Параметр Тип Описание Пример
Название Текст Произвольное имя для идентификации Grid BTC/USDT #1
Пара Выбор Торговая пара (BTC, ETH, XRP и т.д.) BTC/USDT
Режим Выбор Demo (тестовая) или Live (реальная) Demo
Начальный баланс Число Сумма средств для начала торговли (USDT) 1000
Кол-во уровней Число Для Grid: число сеток распределения 10
Верхняя цена Число Максимальная цена диапазона 50000
Нижняя цена Число Минимальная цена диапазона 40000
Stop Loss % Число Процент убытка для автоматического закрытия 5

4️⃣ Состояния стратегии

Каждая стратегия находится в одном из следующих состояний:

🟢 Активна

Торговля идёт, бот выполняет ордера

Действие: Остановить

🔴 Остановлена

Ждёт команды на запуск

Действие: Запустить

📦 Архив

Бывшая активная стратегия, история

Действие: Восстановить

⚠️ Ошибка

Произошла критическая ошибка

Действие: Проверить логи

5️⃣ Просмотр аналитики

Каждая стратегия предоставляет детальную аналитику:

Общая информация

  • Общий профит/убыток: В USD и в процентах от вложения
  • Количество сделок: Всего закрытых ордеров
  • Win rate: Процент прибыльных сделок
  • Средний профит: Среднее значение профита на сделку

Информация о рисках

  • Плечо/маржа: Если используется маржинальная торговля
  • Максимальный убыток: Самая убыточная сделка
  • Открытые позиции: Количество активных ордеров
  • Уровень риска: Общая оценка риска стратегии

Просмотр деталей

Нажмите на название стратегии для раскрытия полного списка всех связанных ордеров, логов ошибок и событий выполнения.

6️⃣ Безопасность администратора

🔒 Важные требования безопасности

  • Доступ: Админ-панель доступна только авторизованным администраторам
  • Пароль: Используйте надёжный пароль (минимум 15 символов)
  • 2FA: Рекомендуется включить двухфакторную аутентификацию
  • Логирование: Все административные действия записываются в логи
  • IP адреса: По возможности ограничивайте доступ определёнными IP

Что логируется

  • Создание и редактирование стратегий
  • Запуск и остановка торговли
  • Изменение критических параметров
  • Вход в админ-панель
  • Экспорт данных

Рекомендации

  • Не делитесь логин/паролем с другими пользователями
  • Регулярно проверяйте логи для обнаружения подозрительной активности
  • Используйте сильные пароли и периодически их меняйте
  • Ограничьте число администраторов минимально необходимым
